# CryptoID Mobile - Decentralized Identity Wallet

A mobile application for self-sovereign identity using blockchain technology.

## Features
- **DID Creation**: Create decentralized identifiers (DID) on Ethereum/Polygon
- **Wallet Generation**: Secure hierarchical deterministic wallet generation
- **Credential Management**: Issue, store, and verify verifiable credentials
- **QR Sharing**: Share credentials via QR codes with selective disclosure
- **Zero-Knowledge Proofs**: Privacy-preserving verification (placeholder)
- **Secure Vault**: Encrypted local storage with biometric protection

## Architecture
- **Frontend**: React Native with Expo
- **Blockchain**: Ethereum/Sepolia (with Polygon support)
- **Smart Contracts**: Solidity identity registry
- **Storage**: Secure enclave + IPFS for credentials

## Getting Started

### Prerequisites
- Node.js 16+
- Expo CLI
- MetaMask or Ethereum wallet (for testing)
- Infura account (for blockchain access)

### Installation
1. Clone repository:
   ```bash
   git clone https://github.com/yourusername/cryptoid-mobile.git
   cd cryptoid-mobile
```
